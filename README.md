# ExGlass-System
ExGlass/ExPhans system made for BD Stem Stars 2025 competition, and iterated upon and updated for ISE Submission

**NOTE: Treat some parts of this code as cleaned portfolio excerpts based on real code, some part of the code is not the entire production program.**

- Arduino's example sketch code "CameraWebServer" was used as a starting reference for certain ESP32 and ESP8266 onboard code, and adapted for this project.
  This can be found by simply going to File > Examples > ESP32 > Camera > CameraWebServer within Arduino's V1.8.19 program.
